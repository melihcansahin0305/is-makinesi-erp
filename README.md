# İş Makinesi ERP

İş makinesi firması için ERPNext üzerine inşa edilmiş kapsamlı ERP çözümü.

## Özellikler

- Makine yönetimi ve takibi
- Personel ve operatör yönetimi
- Makine ve personel puantajı
- Makine ve personel hakedişi
- Yakıt takibi
- Bakım ve masraf takibi
- Cüruf işleme ve skal üretimi
- Kantar fişi ve tartım takibi
- Skal satış ve stok takibi
- Kâr-zarar analizi ve raporlama
- Bildirim ve uyarı sistemi

## Gereksinimler

- Frappe Framework v15+
- ERPNext v15+
- Python 3.11+

## Kurulum

```bash
bench get-app is_makinesi_erp
bench --site [site-name] install-app is_makinesi_erp
```
